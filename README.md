## Installation

1. Clone this repository to your local machine using:
   ```bash
   git clone https://github.com/xyzuan/xyzuanFtReact
   ```
2. Navigate to the project directory:
   ```bash
   cd xyzuanFtReact
   ```
3. Install the dependencies using your preferred package manager:
   ```bash
   npm install
   # or
   yarn install
   ```
